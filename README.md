### Você é Nelso, um gato estudante de Ciência da Computação que precisa urgentemente entregar o trabalho de Computação Gráfica. Entretando, nem tudo é tão fácil. Antes que seja tarde demais, pegue as duas metades da sua carteirinha de estudante que foi quebrada ao meio, pegue seu notebook, entre no ônibus e apresente seu trabalho em sala de aula.


# Instalação
## 1. Selecione a pasta onde quer salvar o jogo

## 2. Abra o terminal na pasta selecionada

### > git clone https://github.com/crispimUchoa/CatVsComputerGraphics.git

### > cd ./CatVsComputerGraphics

### > python3 -m venv venv

### No linux: > source venv/bin/activate
### No windows: > venv/Scripts/activate

### > pip install -r requirements.txt

### > python main.py

### 3. Divirta-se
