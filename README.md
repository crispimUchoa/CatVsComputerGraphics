Antes que seja tarde demais, pegue as duas metades da sua carteirinha de estudante que foi quebrada ao meio, pegue seu notebook, entre no ônibus e apresente seu trabalho em sala de aula.


# Instalação
## 1. Selecione a pasta onde quer salvar o jogo

## 2. Abra o terminal na pasta selecionada

## 3. Execute git clone https://github.com/crispimUchoa/CatVsComputerGraphics.git

## 4. Execute cd ./CatVsComputerGraphics

## 5. Execute python3 -m venv venv

## 6. Execute:
##    No linux: source venv/bin/activate
##    No windows: venv/Scripts/activate

## 7. Execute pip install -r requirements.txt

## 8. Exceute python main.py

## 9. Divirta-se
